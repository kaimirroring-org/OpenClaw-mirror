El modelo Qwen3-14B-Q5_K_M está instalado en la PC gaming local (RTX 5070 Ti) y se utiliza como 'cerebro' para el VPS OL9 con OpenClaw. Este VPS actúa como agente autónomo que consume el modelo local a través de la red. Esta configuración permite una ejecución eficiente de OpenClaw con acceso al modelo de alta capacidad en la infraestructura local.
